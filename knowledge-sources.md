# Knowledge Sources

This file lists the sources used by the Open Research Workflow Agent to surface relevant guidance and signpost researchers to appropriate resources. The agent will only reference sources included here — it does not draw on outside knowledge. The list is intentionally curated rather than exhaustive, prioritising reputable, widely-used resources relevant to UK research practice. It is designed to be configurable: deployers are encouraged to add or substitute sources appropriate to their institutional context, provided those sources are publicly available.
